# Task Guide

Find documentation by what you want to accomplish.

## Quick Reference

| I want to... | Key | Guide |
|--------------|-----|-------|
| Move between windows | `Ctrl+h/j/k/l` | [Navigation](navigation.md) |
| Switch buffers | `bn` / `bp` | [Navigation](navigation.md) |
| Go to definition | `gd` | [Navigation](navigation.md) |
| Jump back | `Ctrl+o` | [Navigation](navigation.md) |
| Comment code | `gcc` | [Editing](editing.md) |
| Surround with quotes | `ysiw"` | [Editing](editing.md) |
| Trigger completion | `Ctrl+Space` | [Editing](editing.md) |
| Rename symbol | `<leader>cr` | [Code Intelligence](code-intelligence.md) |
| Format code | `<leader>cf` | [Code Intelligence](code-intelligence.md) |
| See diagnostics | `<leader>xx` | [Code Intelligence](code-intelligence.md) |
| Find files | `<leader>ff` | [Searching](searching.md) |
| Search in files | `<leader>fg` | [Searching](searching.md) |
| Stage hunk | `<leader>hs` | [Git](git.md) |
| View blame | `<leader>hb` | [Git](git.md) |
| Set breakpoint | `<leader>db` | [Debugging](debugging.md) |
| Start debugging | `<leader>dc` | [Debugging](debugging.md) |

## Leader Key Groups

Press `<Space>` (leader) and wait for which-key to show options.

| Prefix | Category | Guide |
|--------|----------|-------|
| `<leader>b` | Buffer | [Navigation](navigation.md) |
| `<leader>c` | Code | [Code Intelligence](code-intelligence.md) |
| `<leader>d` | Debug | [Debugging](debugging.md) |
| `<leader>f` | Find | [Searching](searching.md) |
| `<leader>h` | Git Hunks | [Git](git.md) |
| `<leader>t` | Toggle | Various |
| `<leader>x` | Diagnostics | [Code Intelligence](code-intelligence.md) |
